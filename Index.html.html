¿Cómo compartir este aplicativo con los estudiantes?

Guía de despliegue · Solo visible para el docente

✅ Opción 1 — Google Drive (la más rápida)

  1. Sube este archivo .html a tu Google Drive.
  2. Haz clic derecho → Compartir → cambia el acceso a "Cualquier persona con el enlace".
  3. Copia el enlace y compártelo por el chat de Moodle, WhatsApp o correo.
  4. Los estudiantes lo abren directamente en el navegador. No necesitan cuenta de Google.
⚠ Limitación: Google Drive a veces no renderiza HTML directamente. Si pasa eso, usa la Opción 2.

✅ Opción 2 — GitHub Pages (gratuito y permanente, recomendado)

  1. Crea una cuenta gratuita en github.com.
  2. Crea un repositorio nuevo (puede ser privado o público).
  3. Sube este archivo y renómbralo index.html.
  4. Ve a Settings → Pages y activa GitHub Pages desde la rama main.
  5. GitHub te dará una URL del tipo tuusuario.github.io/detective-sesgos que puedes pegar en Moodle.

✅ Opción 3 — Incrustar en Moodle (actividad "Página")

  1. En tu curso de Moodle, activa la edición y añade una actividad de tipo "Página".
  2. En el editor de la página, haz clic en el botón HTML (icono <>).
  3. Pega todo el contenido de este archivo HTML.
  4. Guarda. Moodle renderizará el aplicativo directamente dentro del curso.
⚠ Limitación importante: el registro (localStorage) funciona por navegador y dispositivo. Tú no verás los registros desde tu equipo — solo el estudiante los ve en su navegador. Para registro centralizado, usa la Opción 2 con exportación JSON, y pídeles que te envíen el archivo exportado, o usa la integración con una base de datos externa.

📋 Flujo recomendado para evaluación formal

  1. Publica el aplicativo con la Opción 2 (GitHub Pages).
  2. Abre el aplicativo en el computador del salón al iniciar la sesión de evaluación.
  3. Cada estudiante hace su intento desde su propio dispositivo o desde el computador.
  4. Al terminar todos, haz clic en "Exportar JSON" en el panel docente para descargar el registro completo.
  5. Ese archivo JSON contiene nombre, cédula, fecha, hora, correctas y nota de cada estudiante.
